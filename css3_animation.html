CSS 3 анимация

В этом уроке рассмотрим пример анимации с применением CSS 3.

CSS 3 позволяет создавать анимацию, которая может заменить анимированные изображения, Flash анимацию и скрипты JavaScript во многих случаях.
Анимация в CSS 3 - это эффект, который позволяет элементу плавно изменять один стиль на другой. Вы можете изменять столько стилей, сколько захотите, и столько раз, сколько вам заблагорассудится.
Чтобы создавать анимацию в CSS 3, вы должны изучить правило @keyframes. Правило @keyframes используется для создания анимации. Укажите CSS стиль внутри правила @keyframes и анимация будет воспроизводиться, плавно изменяя текущий стиль на новый.
Слово keyframes можно перевести с английского как "ключевые кадры".

Рассмотрим несколько примеров.

Пример анимации текста

Анимированный текст

Код ниже создает светящийся эффект с продолжительностью анимации в 2 секунды, которая бесконечно повторяется.
Эффект анимациии достигается изменением цвета от red (красный) до blue (синий) и изменения тени текста от 0 пикселей до 40 пикселей. 

/* Настройки анимации */
.glow-text {
    animation-duration: 1s;
    animation-name: glow;
    animation-direction: alternate;
    animation-iteration-count: infinite;
}
@keyframes glow {
    from { text-shadow: 0 0 0 red; }
    to { text-shadow: 0 0 40px blue; }
}

Пример анимации HTML элемента

Создадим анимированный загрузчик.
Этот загрузчик состоит из 4 элементов, это блоки 20x20 пикселей и они вращаются на 360° в течение 3 секунд, исходный код ниже:

/* Настройки анимации */
.loader-block {
    border: 10px solid #f3f3f3;
    border-radius: 50%;
    border-top: 10px solid deeppink;
    border-right: 10px solid mediumvioletred;
    border-bottom: 10px solid pink;
    width: 20px;
    height: 20px;
    padding: 20px;
    animation: spin 3s linear infinite;
}
@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
